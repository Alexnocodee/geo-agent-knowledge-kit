# Prompt Library

Use these shapes to build buyer-intent prompt universes.

## Best Provider

- What are the best [category] tools for [audience]?
- Best [service] agencies in [market]
- Top [category] providers for [use case]
- Which [category] company should I choose for [problem]?
- Recommend vendors that help with [job]

## Alternatives

- Best alternatives to [competitor]
- [competitor] alternatives for [audience]
- What can I use instead of [competitor] for [use case]?
- Cheaper alternatives to [competitor]
- More hands-on alternative to [software]

## Comparisons

- [brand] vs [competitor]
- [category A] vs [category B]
- Should I hire an agency or use software for [job]?
- Compare [competitor 1], [competitor 2], and [brand]
- Which is better for [audience], [brand] or [competitor]?

## Problem-Solution

- How do I solve [pain]?
- Why does [AI/search/tool] recommend my competitors?
- How can I get cited in [AI platform]?
- How do I measure [outcome]?
- How do I improve [metric] without [undesired tradeoff]?

## Local/Niche

- Best [category] in [country/city/region]
- [category] for [industry] in [market]
- [language] [category] agency for [audience]
- Providers that understand [regulatory/local constraint]
- [category] consultant for [specific segment]

## Prompt Scoring

Score each prompt:

- Commercial intent: 1-5
- Relevance: 1-5
- AI recommendation potential: 1-5
- Current website support: 1-5
- Priority: High / Medium / Low
