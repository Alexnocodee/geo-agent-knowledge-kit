# Measurement Setup

Use this when the client asks how to track GEO or AI visibility.

## Minimum Setup

- Prompt matrix
- Manual run log
- Competitor mention log
- Citation/source log
- GA4 referral review
- Landing page and conversion tracking

## Prompt Matrix Columns

- Date
- Market
- Language
- Platform
- Prompt
- Run number
- Brand mentioned
- Brand cited
- Competitors mentioned
- Sources cited
- Sentiment
- Notes

## GA4 Checks

Look for referrals from:

- chatgpt.com
- chat.openai.com
- perplexity.ai
- gemini.google.com
- copilot.microsoft.com
- claude.ai
- you.com

The exact referral patterns may change. Verify in the client's analytics.

## Reporting

Report:

- Mention rate
- Citation rate
- Recommendation rate
- Competitor share
- Source/citation share
- AI referral sessions
- AI referral conversions
- Content/citation changes made
