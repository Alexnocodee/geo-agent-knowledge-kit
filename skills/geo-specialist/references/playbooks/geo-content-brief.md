# GEO Content Brief

Use this to create a page brief that is useful to humans and extractable by AI systems.

## Inputs

- Target prompt
- Page type
- Audience
- Product/service
- Competitors or alternatives
- Required proof
- Target market/language

## Brief Structure

1. Page goal
2. Target buyer prompts
3. Search/AI intent
4. Recommended title and H1
5. Required sections
6. Answer blocks
7. Tables or comparison matrices
8. Proof requirements
9. FAQ block
10. Schema recommendations
11. Internal links
12. External citation targets
13. Measurement plan

## GEO Writing Rules

- Put the direct answer high on the page.
- Define entities and categories clearly.
- Use concrete claims.
- Support claims with proof.
- Include comparison logic.
- Avoid vague superlatives.
- Make content easy to quote.
