# AI Search Systems

## Main Surfaces

GEO work usually targets:

- ChatGPT with web search
- Perplexity
- Google AI Overviews and AI Mode
- Gemini
- Microsoft Copilot
- Claude with web/search features when available
- Vertical AI assistants in commerce, travel, finance, legal, and B2B software

## Simplified Retrieval Pipeline

Most AI answer systems follow a rough pattern:

1. Interpret the user prompt.
2. Break it into subqueries or retrieval needs.
3. Retrieve candidate sources from search indexes, web crawlers, knowledge graphs, or partner data.
4. Rerank or filter sources.
5. Generate a synthesized answer.
6. Attach citations or source references when the product supports them.

The exact pipeline differs by platform and changes over time.

## What This Means For GEO

The page does not only need to "rank". It must be useful at several moments:

- Retrieval: the system can find it.
- Reranking: the page looks relevant and authoritative enough.
- Extraction: the answer can pull clean facts and claims from it.
- Generation: the brand can be described accurately.
- Citation: the page is suitable to show as a source.

## Platform Differences

Do not assume all AI systems behave the same.

- Perplexity often behaves closer to a citation-heavy answer/search product.
- Google AI Overviews are tied to Google's search systems and can vary by query type.
- ChatGPT may cite fewer sources but can still absorb source content into the answer.
- Gemini and Copilot may draw from different search integrations and knowledge sources.

## Non-Determinism

AI answers vary across:

- Runs
- Prompt wording
- User location
- Language
- Time
- Model version
- Retrieval freshness

Therefore, one prompt check is not a reliable measurement. Use repeated prompt tests, prompt clusters, and confidence bands when possible.

## Multilingual Markets

For Europe, do not audit only in English. Query language can change which brands are recommended and how they are described.

Recommended approach:

- Test English plus the main local language.
- Include local competitors and local third-party sources.
- Build pages and citations for high-value languages and markets.
