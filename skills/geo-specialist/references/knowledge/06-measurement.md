# Measurement

## Why Measurement Is Hard

AI visibility is probabilistic. Answers vary by prompt, run, model, language, market, and time.

Do not use one answer as a stable metric.

## Core Metrics

- Prompt coverage: how many important prompts the brand has content for.
- Mention rate: how often the brand appears.
- Citation rate: how often the brand URL is cited.
- Recommendation rate: how often the brand is suggested as an option.
- Position or prominence: where the brand appears in the answer.
- Sentiment/accuracy: how the brand is described.
- Competitor share: which competitors appear more often.
- Source share: which third-party sources influence answers.
- AI referral traffic: sessions from AI surfaces.
- Conversion quality: leads, signups, revenue, or pipeline from AI traffic.

## Minimum Manual Test

If no platform/tool is available:

1. Build 20-50 buyer prompts.
2. Test across 2-3 AI/search surfaces where allowed.
3. Repeat each important prompt multiple times.
4. Record brand mention, competitors, citations, sentiment, and sources.
5. Mark results as directional.

## Better Test

Use a prompt matrix:

- Prompt
- Prompt type
- Market
- Language
- AI platform
- Run number
- Brand mentioned
- Brand cited
- Competitors mentioned
- Sources cited
- Sentiment
- Notes

## Analytics Setup

Check:

- GA4 referral traffic from AI domains
- UTM usage in campaigns
- Server logs for AI crawlers where accessible
- Landing pages receiving AI referrals
- Conversion events for AI-referred sessions
- CRM source mapping for high-intent leads

## Reporting Rule

Report confidence:

- High: repeated live tests plus analytics or source evidence.
- Medium: website plus SERP/source evidence.
- Low: strategic inference only.
