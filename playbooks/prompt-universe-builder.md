# Prompt Universe Builder

Use this to build the set of AI-search prompts a brand should be visible for.

## Inputs

- Domain
- Product/service
- Audience
- Market/language
- Competitors
- Use cases
- Pains solved

## Prompt Categories

Build at least 40 prompts for full work, or 20 for a mini-audit:

- Best provider
- Alternatives
- Comparisons
- Problem-solution
- How-to
- Buying criteria
- Local/niche
- Integration/use-case
- Industry-specific
- Risk/objection

## Scoring

Score:

- Commercial intent
- Relevance
- Recommendation potential
- Current coverage
- Competitive difficulty
- Priority

## Rule

Prompts should sound like real buyer questions, not SEO keywords pasted into a prompt box.
